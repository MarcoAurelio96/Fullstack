# 🏋️‍♂️ Iron Pace

> **Supera tus límites, registra tus progresos.**
> Iron Pace es una aplicación de fitness diseñada para atletas que necesitan registrar sus entrenamientos de fuerza y cardio de la forma más rápida y cómoda posible, directamente desde el gimnasio.

## ✨ Características Principales

Iron Pace está pensada para usarse con una sola mano mientras entrenas. Olvídate de interfaces complicadas:

*   📱 **Diseño 100% Mobile-First:** Navegación inferior intuitiva (Bottom Bar) y modales ergonómicos (Bottom Sheets) adaptados a pantallas de móviles.
*   💪 **Bibliotecas Personalizadas:** Crea, edita y guarda tus ejercicios de Gym y tus rutas de Cardio habituales.
*   ⏱️ **Sesiones en Vivo:** Inicia un entrenamiento, marca tus series completadas y utiliza el cronómetro integrado para controlar tus descansos sin salir de la app.
*   🏆 **Gamificación:** Recibe medallas diarias y celebra tus victorias con confeti al completar tu rutina.
*   📅 **Historial Detallado:** Revisa qué días entrenaste, cuántos ejercicios hiciste y qué pesos moviste.
*   👤 **Perfil y Estadísticas:** Calcula tu IMC al instante y lleva la cuenta total de tus sesiones de sudor y esfuerzo.

## 🛠️ Tecnologías Utilizadas

Este proyecto es una aplicación Fullstack que sigue una arquitectura moderna:

**Frontend:**
*   [React](https://reactjs.org/) + [Vite](https://vitejs.dev/) - Renderizado ultrarrápido.
*   [TypeScript](https://www.typescriptlang.org/) - Tipado estricto para evitar errores.
*   [Tailwind CSS](https://tailwindcss.com/) - Estilos ágiles, modernos y completamente adaptables (Responsive).
*   [Firebase](https://firebase.google.com/) - Autenticación segura de usuarios.
*   [Lucide React](https://lucide.dev/) - Iconografía limpia y profesional.

**Backend & Red:**
*   [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/) - Servidor API RESTful.
*   Arquitectura por capas (Rutas, Controladores, Servicios).
*   Cliente de red tipado para asegurar que los datos del frontend y backend coinciden perfectamente.
