# Organización del Trabajo

Para este proyecto utilizamos la metodología **Kanban** mediante la herramienta **Trello**. 

### Flujo de las columnas:
* **Backlog**: Tareas futuras y nuevas ideas.
* **Todo**: Tareas listas para ser desarrolladas.
* **In Progress**: Tareas en desarrollo activo.
* **Review**: Tareas terminadas que necesitan pruebas.
* **Done**: Funcionalidades completadas y funcionando.
